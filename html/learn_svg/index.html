SVG学习笔记

HTML标签

在HTML文件中，可通过<embed>、<object>、<iframe>标签将外部SVG文件嵌入到文档，也可直接使用<svg>标签在文档中嵌入代码。

<svg>
  <circle cx="100" cy="50" r="40" stroke="black" stroke-width="2" fill="red"/>
</svg>

预定义形状

SVG有一些预定义的形状元素，可被开发者使用和操作：

  1. 矩形 <rect>

    <svg>
  <rect width="200" height="100" style="fill:rgb(0,0,255);stroke-width:1;stroke:rgb(0,0,0)"/>
</svg>

  2. 圆形 <circle>

    <svg>
    <circle cx="100" cy="50" r="40" stroke="black" stroke-width="2" fill="red"/>
<svg>

  3. 椭圆 <ellipse>

    <svg>
    <ellipse cx="100" cy="80" rx="100" ry="50" style="fill:yellow;stroke:purple;stroke-width:2"/>
</svg>

  4. 线段 <line>

    <svg>
    <line x1="0" y1="0" x2="200" y2="200" style="stroke:rgb(255,0,0);stroke-width:2"/>
</svg>

  5. 多段线 <polyline>

    <svg>
    <polyline points="20,20 60,25 160,40 180,30 200,10" style="fill:none;stroke:black;stroke-width:3" />
</svg>

  6. 多边形 <polygon>

    <svg>
    <polygon points="200,10 300,50 50,50" style="fill:lime;stroke:purple;stroke-width:1"/>
</svg>

  7. 文本 <text>

    <svg>
  <text x="0" y="15" fill="red">iknot.org</text>
</svg>

路径 <path>

元素用于定义一个路径,下面的命令可用于路径数据。所有命令均允许小写字母。大写表示绝对定位，小写表示相对定位。

  • M = moveto
  • L = lineto
  • H = horizontal lineto
  • V = vertical lineto
  • C = curveto
  • S = smooth curveto
  • Q = quadratic Bézier curve
  • T = smooth quadratic Bézier curveto
  • A = elliptical Arc
  • Z = closepath
<svg>
  <g fill="none" stroke="black" stroke-width="2">
    <path stroke-dasharray="5,5" d="M5 20 L215 20" />
    <path stroke-dasharray="10,10" d="M5 40 L215 60" />
  </g>
</svg>

Stroke属性（线型）

  • stroke:颜色
  • stroke-width：线宽
  • stroke-linecap：端部形状
  • stroke-dasharray：线型数据（实虚线）

下列数据20,10,5,5,5,10依次定义了实、虚、实、虚、实、虚部分的长度

<path stroke-dasharray="20,10,5,5,5,10" d="M5 60 l215 0" />

参考

SVG和canvas的区别:

Canvas:

依赖分辨率
不支持事件处理器
弱的文本渲染能力
能够以 .png 或 .jpg 格式保存结果图像
最适合图像密集型的游戏，其中的许多对象会被频繁重绘

SVG:

不依赖分辨率
支持事件处理器
最适合带有大型渲染区域的应用程序（比如谷歌地图）
复杂度高会减慢渲染速度（任何过度使用 DOM 的应用都不快）
不适合游戏应用
