基本概念

Data Point(Argument,Value)

data point是图表的最基本元素,一组data point组成一个Series. DevExpress根据Series的view type来处理和显示其中的数据。

Series,Series Views

Diagram

Diagram是chart的主要工作区域。可在设计时指定Diagram类型，也可在运行时使用类型转换获得Diagram的 的不同类型。XYDiagram是最常用的类型。 {% codeblock lang:csharp%} XYDiagram diagram = chart.Diagram as XYDiagram; if (diagram != null) { diagram.Rotated = true; diagram.EnableAxisXScrolling = true; } {% endcodeblock %}

Axes

Axes表示图表的坐标轴，为数据点的augument和value提供坐标系。

更改坐标轴颜色

{% codeblock lang:csharp%} XYDiagram diagram = chart.Diagram as XYDiagram; if (diagram != null) { diagram.AxisX.Color = Color.FromArgb(237, 125, 49); } {% endcodeblock %}

Panes

Pane是Diagram中的画图区。只有XYDiagram2D右Panes属性。

Series View

Series View分为2D和3D视图，用来控制数据的表现方式。一个chart中可以包含多种Series View。
